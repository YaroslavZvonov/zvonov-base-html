HTML-документ - это обычный текстовый документ, может быть создан как в обычном текством редакторе (Блокнот), так и в специализированном, с подсветкой кода (Notepad++, Visual Studio Code и т.п). HTML-документ имеет расширение .html.

HTML-документ состоит из дерева HTML-элемента и текста. Каждый элемент обозначается в исходном документе начальным (открывающим) и конечным (закрывающи) тегом (за редким исключением).

Начальный тег показывает, где начинается элемент, конечный - где заканчивается. Закрывающий тег образуется путем добавления слэша / перед именем тега: <имя тега > ... </имя тега>. Между начальным и закрывающий тегами находится содержимое элемента - контент.

Например:

1. <p> Контент </p> - Создает параграф

2. <p> Контент </p> - Делает полужирное начертание

3. <p> Контент </p> - Курсивное начертание
